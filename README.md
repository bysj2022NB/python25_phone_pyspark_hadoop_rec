## 介绍
基于知识图谱的图神经网络&图注意力推荐算法加持
 pytorch+neo4j+GNN+GAE模型实现推理
 (本人手写实现模型算法，算法代码1万行，吊打清华北大斯坦福)
本项目旨在通过Python技术栈对京东平台上的手机数据进行抓取、分析并构建一个简单的手机推荐系统。
主要功能包括：
网络爬虫：从京东获取手机数据；
数据分析：统计各厂商手机销售分布、市场占有率、价格区间和好评率；
可视化展示：使用ECharts进行数据可视化；
推荐系统：根据分析结果为用户推荐手机。
